## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/hackermohan12/getidcard.git
```

2. Navigate to the project directory:
```bash
cd getidcard
```

3. Open `index.html` in your web browser:
```bash
# On macOS
open index.html

# On Linux
xdg-open index.html

# On Windows
start index.html
```

Or simply drag and drop the `index.html` file into your browser.

## Usage

1. **Select a Template**: Choose from available ID card templates
2. **Upload Photo**: Click to upload your photo (JPG, PNG supported)
3. **Fill Details**: Enter the required information:
   - Full Name
   - Employee/Student ID
   - Department/Position
   - Contact Information
   - Other relevant fields
4. **Preview**: View your ID card in real-time
5. **Download**: Click the download button to save your ID card

## Customization

### Adding New Templates

1. Create your template design in the `images/templates/` directory
2. Update the template selector in `index.html`
3. Add corresponding CSS styles if needed

### Modifying Fields

Edit the form fields in `index.html` to add or remove information fields according to your requirements.
